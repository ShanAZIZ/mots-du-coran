## Specifications V2
- Tout doit etre compatible Laptop et Mobile
- Page d'acceuil
    - Ajouter un bouton "Le Coran"
- Page Coran
    - Lister les sourates
    - Au click sur une sourate, on redirige vers la page Sourate
    - Barre de recherche sourates
    - Boutons rechercher dans tout les coran (Rerigire vers la page recherche avancée)

- Page Sourate
    - Afficher les versets du sourates
    - Bouton navigation vers un verset


- Page Mots
    - Boutons sur chaque mots : Rechercher dans le coran -> Redirige vers page recherche avancée préremplie avec le mot + Traduction

- Page recherche avancée
    - Barre de recherche arabe
    - Barre de recherche traduction
    - Surligner l'occurence de la recherche dans les sourates
    - Card de resultats (Verset + Sourate)
